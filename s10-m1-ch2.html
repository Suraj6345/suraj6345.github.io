Q.2 Decide which of the following are quadratic equations.

(1) x² + 5x - 2 = 0

Here, x is the only variable and
the maximum index of the variable is 2.
∴ It is a quadratic equation.

(2) y² = 5y - 10

y² - 5y + 10 = 0
Here, y is the only variable and
the maximum index of the variable is 2.
∴ It is a quadratic equation.

(3) y² + 1/y = 2

Multiplying by y:
y³ + 1 = 2y
y³ - 2y + 1 = 0
The maximum index is 3 (not 2).
∴ It is NOT a quadratic equation.

(4) x + 1/x = -2

Multiplying by x:
x² + 1 = -2x
x² + 2x + 1 = 0
Maximum index is 2.
∴ It is a quadratic equation.

(5) (m + 2)(m - 5) = 0

m(m - 5) + 2(m - 5) = 0
m² - 5m + 2m - 10 = 0
m² - 3m - 10 = 0
∴ It is a quadratic equation.

(6) m³ + 3m² - 2 = 3m³

3m² - 2 = 3m³ - m³
2m³ - 3m² + 2 = 0
The maximum index is 3.
∴ It is NOT a quadratic equation.

Q.4 Determine if given values are roots:

(1) x² + 4x - 5 = 0 ; x = 1, -1

LHS = x² + 4x - 5
For x = 1: (1)² + 4(1) - 5 = 1 + 4 - 5 = 0
LHS = RHS. ∴ 1 is a root.

For x = -1: (-1)² + 4(-1) - 5 = 1 - 4 - 5 = -8
LHS ≠ RHS. ∴ -1 is NOT a root.

(2) 2m² - 5m = 0 ; m = 2, 5/2

For m = 2: 2(2)² - 5(2) = 8 - 10 = -2. NOT a root.
For m = 5/2: 2(5/2)² - 5(5/2) = 2(25/4) - 25/2
= 25/2 - 25/2 = 0. IS a root.
Q.5 Find k if x = 3 is a root of kx² - 10x + 3 = 0
Substituting x = 3 in the equation:
k(3)² - 10(3) + 3 = 0
9k - 30 + 3 = 0
9k - 27 = 0
9k = 27
k = 3
Value of k = 3

Practice Set 2.2
1) x² - 15x + 54 = 0
Logic: Multiply to 54, Add to -15. Numbers: -9, -6
x² - 9x - 6x + 54 = 0
x(x - 9) - 6(x - 9) = 0
(x - 9)(x - 6) = 0
x - 9 = 0 OR x - 6 = 0
x = 9 OR x = 6
Roots are 9 and 6.
2) x² + x - 20 = 0
Logic: Multiply to -20, Add to +1. Numbers: +5, -4
x² + 5x - 4x - 20 = 0
x(x + 5) - 4(x + 5) = 0
(x + 5)(x - 4) = 0
x = -5 OR x = 4
Roots are -5 and 4.
3) 2y² + 27y + 13 = 0
Logic: Multiply (2 × 13) = 26, Add to 27. Numbers: 26, 1
2y² + 26y + 1y + 13 = 0
2y(y + 13) + 1(y + 13) = 0
(y + 13)(2y + 1) = 0
y = -13 OR y = -1/2
4) 5m² = 22m + 15
Standard form: 5m² - 22m - 15 = 0
Numbers: -25, 3 (Product -75, Sum -22)
5m² - 25m + 3m - 15 = 0
5m(m - 5) + 3(m - 5) = 0
(m - 5)(5m + 3) = 0
m = 5 OR m = -3/5
5) 2x² - 2x + 1/2 = 0
Multiply equation by 2: 4x² - 4x + 1 = 0
4x² - 2x - 2x + 1 = 0
2x(2x - 1) - 1(2x - 1) = 0
(2x - 1)(2x - 1) = 0
x = 1/2
6) 6x - 2/x = 1

Numbers: -4, 3 (Product -12, Sum -1)

2x(3x - 2) + 1(3x - 2) = 0
x = 2/3 OR x = -1/2

x(√2x + 5) + √2(√2x + 5) = 0
(√2x + 5)(x + √2) = 0
x = -5/√2 OR x = -√2
8) 3x² - 2√6x + 2 = 0
