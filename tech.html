Behind the Simulation

300x250 AD

WebGL & Three.js

The core of ParticleLab is built on Three.js, a cross-browser JavaScript library used to create and display animated 3D computer graphics in a web browser using WebGL. By leveraging the GPU, we can render up to 30,000 particles at 60 frames per second.

MediaPipe Gesture Detection

We use Google's MediaPipe Hands, a high-fidelity hand and finger tracking solution. It employs machine learning (ML) to infer 21 3D landmarks of a hand from just a single frame, allowing you to manipulate the particle flow with natural movements.

Custom Shaders

To achieve the neon glow and fluid motion, we use GLSL (OpenGL Shading Language). Vertex shaders handle the positioning of each particle based on noise algorithms, while fragment shaders create the soft, glowing aesthetic of the "Point" primitives.
